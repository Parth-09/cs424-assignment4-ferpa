Chicago Areas Analysis

Area Selection

The following visualization displays a specific Chicago Area and the Community Areas that comprise the Area. From the dropdown menu, select a Chicago Area and view the followed Linked visualizations.


Area Statistics

These bar charts display some statical information about certain areas in Chicago. To interact with these charts, click a "Area" in the legend to highlight an area and learn more about a certain area.

Interactive visulization using search method

The following visulization is the distribuion of different requests per ward. In order to get the information of a specific ward, please enter a number in the search box. For example, number 10 will display the information for bar number 10.


The next visualization displays the information for the different zip codes with different service request types. Enter a Zip Code like 60617 to see the distribution of it over the bar chart.
